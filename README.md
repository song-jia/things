## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# 运行测试
npm test

# 生成code coverage报告到文件夹test/coverage
npm coverage
```
