目录结构
```
.
├── bin                   用于命令行执行的代码
├── build                 用于构建工程的script文件
├── client                客户端代码
├── docs                  文档
├── node_modules          npm安装的modules
├── public                静态文件: 图片，javascript, stylesheet
├── routes                express的router handler，相当于MVC中的controller
├── test                  测试case
├── views                 express模版文件
├── README.md             说明文档
├── app.js                express入口文件
├── config.js             配置文件
├── index.html
└── package.json          npm配置文件
```
